Git Commands Reference

Command Description
git init Initialize a new Git repository
git clone [url] Clone a remote repository
git add [file] Stage files for commit
git commit -m "[message]" Commit staged changes with a message
git push Push commits to remote repository
git pull Fetch and merge remote changes
git branch List, create, or delete branches
git checkout [branch] Switch to a different branch
git merge [branch] Merge a branch into current branch
git status Show working directory status
git log View commit history
git diff Show changes between commits or working directory
git reset [file] Unstage files or reset commits
git revert [commit] Create a new commit that undoes changes
git tag [name] Create, list, or delete tags
git stash Temporarily save uncommitted changes
git remote Manage remote repositories
git fetch Fetch updates from remote repository
git config [key] [value] Set Git configuration options
git blame [file] Show who changed each line in a file
git rebase [branch] Reapply commits on top of another branch
git cherry-pick [commit] Apply a specific commit to current branch
git rm [file] Remove a file from repository
git mv [old] [new] Move or rename a file
git show [commit] Display details of a specific commit
git clean -fd Remove untracked files and directories
git reflog Show reference logs of all changes
git bisect Binary search to find a bad commit
git hook Manage Git hooks for automation
git submodule Manage Git submodules
